# cineScope

A lightweight web app for browsing and exploring movies.

## Tech stack
- JavaScript
- HTML
- CSS

## Getting started
1. Clone the repository:
   ```bash
   git clone https://github.com/KhaliqueAhmed/cineScope.git
   ```
2. Open the project folder.
3. Run the app:
   - Open `index.html` in your browser.

## Project structure (typical)
- `index.html` — entry page
- `css/` — styles
- `js/` — app logic
